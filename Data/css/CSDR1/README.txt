The Catalina Surveys Periodic Variable Catalog

~47,000 periodic variables
Data Release: CSDR1
Link: http://nesssi.cacr.caltech.edu/DataRelease/Varcat.html
Paper: ~47,000 periodic variables

1. Catalog
    1.1. File: catalog-original.txt
    1.2. Link: http://nesssi.cacr.caltech.edu/DataRelease/CatalinaVars.tbl
    1.3. To ease reading of catalog with csv, header is changed:
        OLD: # Catalina_Surveys_ID   Numerical_ID  RA_(J2000)...
        NEW: Catalina_Surveys_ID   Numerical_ID  RA_(J2000)...
    1.4. To avoid confusions of RA, DEC units:
        OLD: ...RA_(J2000)     Dec...
        NEW: ...RA_(J2000)     Dec_...

2. Formatted catalog:
    1.1 File: catalog.txt
    1.2. RA and Dec is formatted to degrees to ease cross-match finding.
    1.3. Labels are passed to string (more readable).

3. Photometry
    2.1. File: photometry.csv 
    2.2. Link: http://nesssi.cacr.caltech.edu/DataRelease/AllVar.phot.gz
